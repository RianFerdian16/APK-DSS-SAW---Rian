CARA KUMPUL UAS DSS/SPK
=======================

Isi zip ini sudah dirapikan untuk checklist UAS:
1. Source code aplikasi Streamlit.
2. Dataset CSV minimal 5 alternatif dan 4 kriteria.
3. Modul SAW, TOPSIS, visualizer, dan Supabase handler.
4. Unit test/edge case di folder tests.
5. Laporan PDF di folder laporan.
6. Panduan script video demo di SCRIPT_VIDEO_DEMO.md.

Cara run lokal:
1. Buka terminal di folder dss_railway_supabase.
2. Jalankan: pip install -r requirements.txt
3. Jalankan: streamlit run app.py
4. Klik Hitung Ranking SAW/TOPSIS.

Cara testing:
1. Buka terminal di folder dss_railway_supabase.
2. Jalankan: pytest

Jika Railway dan Supabase sudah pernah dibuat:
1. Tidak perlu membuat Railway project baru.
2. Tidak perlu membuat Supabase project baru.
3. Replace file project lama dengan isi zip ini.
4. Commit dan push ke GitHub yang sudah tersambung ke Railway.
5. Railway akan redeploy otomatis.
6. Environment variable lama tetap dipakai:
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
7. File supabase/schema.sql hanya perlu dijalankan ulang jika tabel dss_history belum pernah ada.

Catatan penting:
- Video demo 3-5 menit tetap perlu direkam manual karena tugas meminta screen recording aplikasi berjalan.
- Saat video, tunjukkan input/upload CSV, ubah bobot what-if, hitung SAW/TOPSIS, grafik, dan download hasil CSV.
