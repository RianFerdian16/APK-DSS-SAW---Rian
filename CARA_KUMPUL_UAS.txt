CARA KUMPUL UAS

1. Pastikan aplikasi sudah memakai studi kasus laptop:
   - Harga = cost
   - Performa = benefit
   - Baterai = benefit
   - Portabilitas = benefit

2. Jalankan aplikasi lokal:
   pip install -r requirements.txt
   streamlit run app.py

3. Jalankan testing:
   pytest

4. Screenshot untuk laporan:
   - Tabel hasil ranking
   - Grafik ranking
   - Grafik bobot
   - Hasil terminal pytest (3 passed)

5. Jika ingin deploy ulang ke Railway tanpa setting ulang:
   git add .
   git commit -m "uas apk rian"
   git push

6. Railway akan redeploy otomatis.

7. Untuk mengganti nama project/service Railway secara permanen:
   buka Railway Dashboard > pilih project > Settings > rename project/service menjadi "uas apk rian".
